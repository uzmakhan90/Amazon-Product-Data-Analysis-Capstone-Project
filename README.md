# Amazon Product Data Analysis Capstone Project

## Project Overview
This capstone project focuses on analyzing Amazon product data to identify trends, customer preferences, pricing patterns, and product performance. The project demonstrates a complete end-to-end data science workflow, including web scraping, data cleaning, exploratory analysis, model building, and dashboard creation.

## Business Problem
E-commerce businesses need to understand product demand, customer preferences, ratings, and pricing strategies to improve sales and customer satisfaction. This project helps identify important product trends and business opportunities using Amazon product data.

## Objectives
- Collect product data using web scraping
- Clean and preprocess raw product data
- Analyze pricing, ratings, and customer reviews
- Identify top-performing and low-performing products
- Build predictive or classification models
- Create visualizations and dashboards
- Generate business recommendations from data insights

## Key Analysis Performed
### Web Scraping
- Extracted product details from Amazon pages
- Collected product names, prices, ratings, reviews, and categories
- Created a structured dataset from raw web data

### Data Cleaning
- Removed missing values and duplicates
- Standardized product names and formats
- Handled inconsistent price and rating values

### Exploratory Data Analysis
- Analyzed product ratings and review trends
- Compared product categories and pricing
- Identified best-selling and highly rated products

### Model Building
- Built predictive or classification models
- Evaluated model performance using suitable metrics
- Identified key factors affecting product success

### Dashboard
- Created an interactive dashboard with KPIs and trend charts
- Included filters, drill-through, and slicers
- Visualized product performance and category trends

## Key Insights
- Product ratings and review counts strongly affect customer purchasing behavior
- Certain product categories perform better than others
- Pricing trends vary significantly across categories
- Dashboard insights help support better business decisions

## Conclusion
This project demonstrates how web scraping, machine learning, and visualization can be used together to generate actionable business insights from Amazon product data.
